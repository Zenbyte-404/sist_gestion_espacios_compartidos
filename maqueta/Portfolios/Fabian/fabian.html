Sobre mi

Como programador en formación, me apasiona transformar ideas en soluciones digitales funcionales y atractivas. Mi enfoque principal se centra en el desarrollo front-end y back-end, combinando la creatividad del diseño con la lógica de la programación. Estoy comprometido con el aprendizaje continuo y con la creación de experiencias de usuario intuitivas y eficientes, siempre buscando integrar las últimas tecnologías para resolver problemas del mundo real.

  • **Desarrollo Frontend:** Creación de interfaces de usuario modernas y responsivas utilizando HTML, CSS y Bootstrap, asegurando una excelente experiencia de usuario en cualquier dispositivo.
  • **Desarrollo Backend:** Construcción de la lógica del servidor y la gestión de datos con Python, garantizando la robustez y escalabilidad de las aplicaciones.
  • **Gestión de Bases de Datos:** Diseño y manipulación de bases de datos relacionales utilizando SQL y MySQL, para una gestión eficiente y segura de la información.
  • **Metodologías de Desarrollo:** Aplicación de principios de desarrollo ágil (como Scrum o Kanban) para gestionar proyectos de forma eficiente, fomentar la colaboración y entregar valor de manera iterativa.
  • **Resolución de Problemas:** Habilidad para analizar desafíos complejos y diseñar soluciones innovadoras, enfocadas en la optimización del rendimiento y la usabilidad.

Sistema de Gestión de Espacios Compartidos

Este proyecto consiste en un sistema web para la gestión de reservas de espacios compartidos, optimizando la organización y el uso de recursos. Desarrollado utilizando HTML, CSS y Bootstrap para una interfaz de usuario intuitiva y responsiva, este trabajo demuestra nuestras habilidades en desarrollo front-end, trabajo en equipo, organización y colaboración.

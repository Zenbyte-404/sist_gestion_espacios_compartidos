## 🎯 Descripción del Proyecto

<div align="center">
  <img src="https://user-images.githubusercontent.com/74038190/212284100-561aa473-3905-4a80-b561-0d28506553ee.gif" width="700">
</div>

**Zenbyte-404** es un **sistema web interactivo** para la gestiónde **espacios compartidos** como:

- 📚 **Salas de estudio** colaborativos
- 💼 **Espacios de coworking** modernos
- 🎯 **Salas de reuniones** empresariales

<div align="center">

### ✨ Características Principales

| 🚀 Funcionalidad | 📝 Descripción |
|:-----------------|:---------------|
| **Reservas Inteligentes** | Sistema automatizado de reservas con calendario interactivo |
| **Gestión de Usuarios** | Control de acceso y perfiles personalizados |
| **Dashboard Analítico** | Métricas en tiempo real del uso de espacios |
| **Notificaciones** | Alertas automáticas y recordatorios |
| **Responsive Design** | Adaptable a todos los dispositivos |

</div>

## 🚀 Instalación y Uso

<div align="center">
  <img src="https://user-images.githubusercontent.com/74038190/212284094-e50ceae2-de86-4dd6-b97c-3b5637362c46.gif" width="300">
</div>

### 📋 Prerrequisitos

- 🌐 Navegador web moderno (Chrome, Firefox, Edge)
- 📱 Conexión a internet
- ⚡ Servidor web local (opcional para desarrollo)

### 🔧 Pasos de Instalación

\`\`\`bash
# 1️⃣ Clonar el repositorio
git clone https://github.com/Zenbyte-404/sist_gestion_espacios_compartidos.git

# 2️⃣ Navegar al directorio
cd sist_gestion_espacios_compartidos

# 3️⃣ Abrir la maqueta en tu navegador
cd maqueta && open index.html

# 4️⃣ Para desarrollo (opcional)
# Instalar extensión Live Server en VS Code
# Click derecho en index.html -> Open with Live Server
\`\`\`

### 🎯 Uso Rápido

1. **🌐 Acceso directo**: [Demo Live](https://zenbyte-404.github.io/sist_gestion_espacios_compartidos/maqueta/index.html)
2. **📱 Navegación**: Explora las diferentes secciones del sistema
3. **🔍 Pruebas**: Interactúa con los formularios y funcionalidades
4. **💻 Desarrollo**: Modifica el código con VS Code para personalizar
